
	1. JDK (KIT DE DESARROLLO DE JAVA)
	
		- SE PUEDE TENER MÁS DE UNA VERSIÓN PERO SÓLO UNA ACTIVA PARA EL SISTEMA.
		- DESCARGAR ÚLTIMA VERSIÓN DESDE WEB OFICIAL DE ORACLE (17.0.2) - VALDRÍA CUALQUIER OTRA EN PRINCIPIO.
			- VERSIÓN EN GOOGLE DRIVE COMPARTIDO.
		- DESCOMPRIMIR Y UBICAR EN UN DIRECTORIO PARA LAS HERRAMIENTAS DE DESARROLLO.
		- CONFIGURAR LAS VARIABLES DE ENTORNO DEL SISTEMA:
			• JAVA_HOME
			• EDITAR VARIABLE PATH PARA AÑADIR %JAVA_HOME%/BIN
		- VERIFICAR EN CMD (CERRAR Y ABRIR EN CASO DE QUE ESTUVIESE ABIERTO) CON EL COMANDO JAVA -VERSION.
		- CUALQUIER ERROR CON EL COMANDO O QUE APAREZCA OTRA VERSIÓN:
			- VERIFICAR VARIABLES DE ENTORNO.
			- ELIMINAR FICHEROS EJECUTABLES DE LA RUTA: C:\Program Files (x86)\Common Files\Oracle\Java\javapath
	
	2. APACHE MAVEN

		- REQUIERE DE JDK INSTALADO PREVIAMENTE.
		- SE PUEDE TENER MÁS DE UNA VERSIÓN PERO SÓLO UNA ACTIVA PARA EL SISTEMA.
		- DESCARGAR ÚLTIMA VERSIÓN DESDE WEB OFICIAL DE APACHE MAVEN (3.8.5) - VALDRÍA CUALQUIER OTRA EN PRINCIPIO.
		- DESCOMPRIMIR Y UBICAR EN UN DIRECTORIO PARA LAS HERRAMIENTAS DE DESARROLLO.
		- CONFIGURAR LAS VARIABLES DE ENTORNO DEL SISTEMA:
			• M2_HOME
			• EDITAR VARIABLE PATH PARA AÑADIR %M2_HOME%/BIN
		- VERIFICAR EN CMD (CERRAR Y ABRIR EN CASO DE QUE ESTUVIESE ABIERTO) CON EL COMANDO MVN -VERSION.
		- CUALQUIER ERROR CON EL COMANDO O QUE APAREZCA OTRA VERSIÓN:
			- VERIFICAR VARIABLES DE ENTORNO.

	3. ECLIPSE
	
		- ES POSIBLE TENER MÁS DE UN ECLIPSE EN EL SISTEMA.
		- DESCARGAR LA VERSIÓN DESDE WEB OFICIAL.
		- DESCOMPRIMIR Y UBICAR EN EL DIRECTORIO CREADO PARA LAS HERRAMIENTAS.
		- CONFIGURAR LA VERSIÓN DEL JDK / APACHE MAVEN EN ENCLIPSE DESDE WINDOWS -> PREFERENCIAS -> JAVA O MAVEN SEGÚN CORRESPONDA.
	
	4. INSTALAR SERVIDOR DE BASE DE DATOS
		- MYSQL SERVER Y/O ORACLE DATABASE.
		- SI MYSQL SERVER:
			- VALE EL INSTALADO A TRAVÉS DE MYSQL WORKBENCH O XAMPP.
		- SI ORACLE DATABASE:
			- FACILITADO EN GOOGLE DRIVE COMPARTIDO.
			- SEGUIR EL INSTALADOR CON LA CONFIGURACIÓN PREDEFINIDA:
				- ESTAR ATENTOS A REUBICAR LA RUTA DE INSTALACIÓN EN NUESTRO DIRECTORIO DE HERRAMIENTAS.
				- CONTRASEÑA "ROOTROOT", USUARIO "SYSTEM".
			- VERIFICAR FICHEROS \dbhomeXE\network\admin\listener.ora y tnsnames.ora
				- SI NO APARECEN O TIENEN ESTRUCTURA ADECUADA DESINSTALAR E INSTALAR DE NUEVO.
				- UTILIZAR EL VALOR DE HOST.
	
	5. INSTALAR GESTOR DE BASE DE DATOS.
		- VÁLIDO PHPMYADMIN O WORKBENCH.
		- RECOMENDADO ORACLE SQL DEVELOPER (EN GOOGLE DRIVE COMPARTIDO)
		- CONFIGURAR CONEXIÓN A ORACLE.
		- CONFIGURAR CONEXIÓN A MYSQL SERVER.
			- REQUIERE CONECTORES (EN GOOGLE DRIVE COMPARTIDO)
	
	6. INSTALAR GIT.
		- DESCARGAR DE LA WEB OFICIAL LA ÚLTIMA VERSIÓN.
		- UTILIZAR EL INSTALADOR CON LA CONFIGURACIÓN POR DEFECTO.
			- ESTAR ATENTOS AL DIRECTORIO DE INSTALACIÓN PARA UBICAR EN NUESTRA CARPETA DE HERRAMIENTAS.
			- SELECCIONAR COMO EDITOR DE CÓDIGO NOTEPAD++, SUBLIMETXT O VISUAL STUDIO CODE.
	
	7. INSTALAR DITTO.
	8. INSTALAR VISUAL STUDIO CODE.
	9. INSTALAR NOTEPAD++ O SUBLIMETXT.
